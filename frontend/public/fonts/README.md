# ChronoTrace self-hosted fonts

These `.woff2` files are committed so the frontend does not depend on third-party font services at runtime.

Source packages:

- `@fontsource/noto-sans-sc@5.2.9`
- `@fontsource/space-grotesk@5.2.10`
- `@fontsource/space-mono@5.2.9`
- `@fontsource/doto@5.2.10`

All listed packages are distributed under `OFL-1.1`.
