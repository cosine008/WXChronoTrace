# 物理标签模板配置操作说明

## 适用场景

表 owner 或管理员需要为某张业务表配置实体物理标签的打印样式时，使用本说明。

适用于以下情况：

- 希望设置批量贴标时默认使用哪种标签模板。
- 希望控制“固定资产”“小标签”“档案封面”哪些模板可用。
- 希望控制标签上打印哪些业务字段。
- 希望隐藏或显示二维码、条码、扫码 URL、标签码、展示码等内容。
- 希望避免敏感字段被打印到实体物理标签上。

当前功能是固定版式的参数化配置，不是自由拖拽式标签设计器。用户可以控制模板、字段和显示项，但不能在页面上自由拖动位置、修改字体或自定义画布。

## 权限要求

只有以下用户可以保存物理标签模板配置：

- 当前表的 owner。
- 系统管理员。

其他角色可以进入表设置查看配置，但不能保存修改。

如果页面显示为只读，通常说明当前用户不是该表 owner，也不是系统管理员。需要联系表 owner 或管理员处理。

## 操作入口

进入目标业务表后，点击：

```text
表设置 -> 物理标签模板
```

也可以直接访问：

```text
/schemas/{schema_id}/settings
```

其中 `{schema_id}` 是当前业务表的 ID。

## 模板类型说明

系统当前支持三种业务标签模板：

| 模板名称 | 模板编码 | 适用场景 | 可打印字段数量 |
|---|---|---|---:|
| 固定资产 | `asset_standard` | 常规资产、设备、物品标签 | 最多 4 个 |
| 小标签 | `small` | 小尺寸贴纸，只需要展示核心信息 | 最多 1 个 |
| 档案封面 | `document_cover` | 档案袋、文件盒、文档封面 | 最多 8 个 |

说明：

- `固定资产` 适合作为默认模板。
- `小标签` 适合空间有限的物理贴纸。
- `档案封面` 适合需要更多字段信息的纸质封面。
- A4 打印只是纸张排版方式，不是这里可配置的业务模板。

## 配置默认模板

在“物理标签模板”面板中找到“默认模板”下拉框。

操作步骤：

1. 打开“默认模板”下拉框。
2. 选择需要作为默认值的模板，例如“固定资产”或“小标签”。
3. 点击“保存”。

默认模板会影响以下行为：

- 批量贴标打开时默认选中的模板。
- 单实体生成标签时，如果没有手动指定模板，会使用该默认模板。
- A4 打印中，如果某些链路需要解析默认标签模板，也会使用该配置。

## 启用或停用模板

每种模板都有“启用”开关。

操作步骤：

1. 在对应模板卡片中找到“启用”。
2. 勾选表示启用该模板。
3. 取消勾选表示停用该模板。
4. 点击“保存”。

启用状态会影响批量贴标中的模板下拉框：

- 已启用模板会出现在下拉框中。
- 已停用模板不会出现在下拉框中。

注意：

- 当前默认模板不能直接停用。
- 如果要停用当前默认模板，需要先把其他模板设为默认，再停用原模板。
- 至少应保留一个可用模板，否则用户无法正常生成新标签。

## 配置打印字段

每个模板都可以选择打印哪些业务字段。

操作步骤：

1. 在目标模板卡片中找到“打印字段”。
2. 勾选需要打印到物理标签上的字段。
3. 取消勾选不需要打印的字段。
4. 点击“保存”。

字段数量限制：

| 模板 | 最多字段数 |
|---|---:|
| 固定资产 | 4 |
| 小标签 | 1 |
| 档案封面 | 8 |

如果选择字段数量超过上限，保存会失败。

## 敏感字段处理

敏感字段不能打印到物理标签上。

页面上如果看到某个字段标记为：

```text
敏感字段不可打印
```

说明该字段不能勾选。

这样做是为了避免身份证号、联系方式、内部编号、保密属性等敏感信息被打印到实体标签上，造成线下泄露风险。

如果确实需要在标签上展示某类信息，建议：

1. 新增一个脱敏后的展示字段。
2. 在业务数据中填入可公开展示的文本。
3. 在物理标签模板中选择这个脱敏字段。

## 配置显示项

每个模板都可以控制以下显示项：

| 显示项 | 说明 |
|---|---|
| 展示码 | 实体面向用户的展示编号 |
| 标签码 | 物理标签自身的唯一编码 |
| QR | 二维码，通常用于扫码进入实体追溯页 |
| Code 128 | 一维条码，可用于扫码枪识别 |
| 扫码 URL | 二维码对应的访问地址文本 |
| 模板标题 | 标签顶部或主要区域的标题信息 |
| 扫码提示 | 提示用户扫码查看实体信息 |

操作步骤：

1. 在目标模板卡片中找到“显示项”。
2. 勾选需要显示的内容。
3. 取消勾选不需要显示的内容。
4. 点击“保存”。

推荐配置：

| 场景 | 推荐显示项 |
|---|---|
| 固定资产 | 展示码、标签码、QR、Code 128、模板标题、扫码提示 |
| 小标签 | 展示码、QR、模板标题 |
| 档案封面 | 展示码、标签码、QR、Code 128、扫码 URL、模板标题 |

## 保存配置

配置修改后，页面右上角会出现未保存状态。

操作步骤：

1. 确认默认模板、启用状态、打印字段和显示项都已配置。
2. 点击“保存”。
3. 等待系统提示“物理标签模板已保存”。
4. 返回当前视图或继续调整其他表设置。

保存成功后，系统会更新该表的配置版本。后续新生成或打印的标签会使用最新配置。

已经下载到本地的旧 SVG 文件不会自动变化。如果需要新样式，需要重新生成或重新下载标签。

## 批量贴标时如何使用配置

进入目标表的当前视图后，点击：

```text
批量贴标
```

批量贴标抽屉中的模板下拉框会自动读取当前表的物理标签模板配置。

使用步骤：

1. 点击“选择当前页”，选中需要贴标的实体。
2. 在模板下拉框中选择模板。
3. 选择下载范围。
4. 点击“生成并下载 A4”。
5. 在确认弹窗中点击“生成并下载”。
6. 系统下载 A4 SVG 文件。

下载范围说明：

| 下载范围 | 说明 |
|---|---|
| 仅本次新生成 | 只下载本次新创建的标签 |
| 本次新生成 + 已有 active | 同时下载新创建标签和已有有效标签 |
| 仅已有 active | 不创建新标签，只下载已有有效标签 |

如果模板下拉框中看不到某个模板，通常说明该模板在表设置中未启用。

## 单实体标签管理时如何使用配置

在当前视图中打开某条实体记录的生命周期抽屉，可以进行单实体标签管理。

常见操作：

- 生成标签。
- 下载或打印单个 SVG 标签。
- 作废 active 标签。
- 替换标签。

单实体生成标签时，如果页面没有显式选择模板，系统会使用当前表的默认模板。

单标签打印时，标签上的字段和二维码、条码等显示项会读取当前表的物理标签模板配置。

## 检查配置是否生效

管理员或表 owner 可以按以下步骤检查：

1. 进入“表设置 -> 物理标签模板”。
2. 将默认模板改为需要测试的模板，例如“小标签”。
3. 保存配置。
4. 回到当前视图。
5. 打开“批量贴标”。
6. 确认模板下拉框默认选中了刚才设置的模板。
7. 选择一条或多条实体，生成并下载 A4 SVG。
8. 打开 SVG 文件，确认字段和显示项符合配置。

重点检查：

- 未启用模板是否从下拉框消失。
- 默认模板是否正确生效。
- 已勾选字段是否出现在 SVG 中。
- 未勾选字段是否没有出现在 SVG 中。
- 敏感字段是否没有出现在 SVG 中。
- QR、Code 128、扫码 URL 等显示项是否符合配置。

## 常见问题

### 看不到“物理标签模板”面板

可能原因：

- 当前页面不是表设置页。
- 当前登录用户没有访问该表的权限。
- 前端页面未刷新到最新版本。

处理方式：

1. 确认进入的是目标表的“表设置”页面。
2. 刷新页面。
3. 如仍不可见，联系管理员确认系统是否已更新到包含物理标签模板配置的版本。

### 保存按钮不能点击

可能原因：

- 当前用户不是 owner 或管理员。
- 当前没有未保存修改。
- 页面正在保存中。

处理方式：

1. 确认自己是否有该表的管理权限。
2. 修改一个配置项后再查看保存按钮。
3. 等待当前保存请求结束后重试。

### 保存失败，提示字段不可打印

可能原因：

- 选择了敏感字段。
- 选择了系统隐藏字段。
- 字段已经从表结构中删除或废弃。

处理方式：

1. 取消勾选敏感字段。
2. 只选择普通业务字段。
3. 如必须展示相关信息，使用脱敏后的公开字段。

### 保存失败，提示字段数量超过模板上限

不同模板有不同字段数量上限：

```text
固定资产：最多 4 个
小标签：最多 1 个
档案封面：最多 8 个
```

减少勾选字段后重新保存。

### 批量贴标中没有想要的模板

可能原因：

- 该模板在“物理标签模板”面板中没有启用。
- 配置还没有保存。
- 页面仍在使用旧缓存。

处理方式：

1. 回到表设置页启用该模板。
2. 点击保存。
3. 回到当前视图并刷新页面。
4. 再次打开批量贴标。

### 修改配置后，旧标签文件没有变化

已经下载到本地的 SVG 文件不会自动更新。

处理方式：

1. 保存新的物理标签模板配置。
2. 重新进入批量贴标或单标签打印。
3. 重新下载 SVG 文件。

## 管理员检查清单

管理员或表 owner 可以按以下顺序排查问题：

1. 确认用户进入的是正确的业务表。
2. 确认用户角色是否为 owner 或系统管理员。
3. 进入“表设置 -> 物理标签模板”。
4. 确认至少一个模板处于启用状态。
5. 确认默认模板也处于启用状态。
6. 检查打印字段是否超过模板上限。
7. 检查是否误选了敏感字段或系统隐藏字段。
8. 保存配置后刷新当前视图。
9. 打开批量贴标，确认模板下拉框和默认模板符合预期。
10. 生成 A4 SVG，检查输出内容是否符合配置。

## 推荐给用户的说明话术

```text
物理标签模板是在表设置里按表配置的。
请进入“表设置 -> 物理标签模板”，选择默认模板，启用需要的模板，勾选要打印的字段和二维码/条码显示项，然后点击保存。
保存后，批量贴标和单实体标签打印都会按这张表的新配置生成 SVG。
敏感字段不能打印到物理标签上，如果需要展示，请改用脱敏后的普通字段。
```

## 相关规则

- 物理标签模板配置是表级配置，不是全局配置。
- 只有 owner 或系统管理员可以保存配置。
- 当前支持“固定资产”“小标签”“档案封面”三种模板。
- 当前不是自由拖拽式模板设计器。
- 敏感字段和系统隐藏字段不能打印到物理标签上。
- 默认模板必须是已启用模板。
- 已下载的旧 SVG 文件不会随配置变更自动更新。
- 新生成或重新打印的标签会使用保存后的最新配置。
